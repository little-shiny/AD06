1. Introducción

En esta tarea obligatoria desarrollarás una aplicación web completa utilizando el patrón MVC (Modelo-Vista-Controlador) con Spring Boot. La aplicación gestionará información de directores y películas de cine.

✅ Tecnologías que utilizarás:
  • Spring Boot: Framework para crear aplicaciones Java
  • Spring Data JPA: Acceso a base de datos con Hibernate
  • Thymeleaf: Motor de plantillas para las vistas HTML
  • MySQL: Base de datos relacional
  • Lombok: Generación automática de getters/setters

El proyecto ya tiene la estructura básica creada. Tu tarea es completar los 50 TODOs marcados en el código para que la aplicación funcione correctamente.

3. Instalación y configuración

Paso 1: Crear la base de datos

Ejecuta el script SQL en MySQL:

mysql -u root -p < mysql/Cine.sql

O abre el archivo mysql/Cine.sql en MySQL Workbench y ejecútalo.

Paso 2: Configurar credenciales

Edita src/main/resources/application.properties con tu usuario y contraseña de MySQL.

Paso 3: Abrir el proyecto

En VS Code: Archivo → Abrir carpeta y selecciona UD06_Obligatorio_RA5

Maven descargará las dependencias automáticamente.

Paso 4: Ejecutar la aplicación

Ejecuta la clase CineApplication.java y accede a http://localhost:8080

4. Descripción del ejercicio

Modelo de datos

Director

  • idDirector: Identificador autogenerado (Long)
  • nombre: Nombre del director (String)
  • anyoNacimiento: Año de nacimiento (int)

Película

  • idPelicula: Identificador autogenerado (Long)
  • titulo: Título de la película (String)
  • anyo: Año de estreno (int)
  • director: Director de la película (relación ManyToOne)
⚠️ Relación entre entidades:

Un director puede dirigir muchas películas (1-N).

Una película tiene un solo director (o ninguno).

@Table(name="Director")
public class Director {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    Long idDirector;
    
    @Column
    String nombre;
    
    @Column
    int anyoNacimiento;
}

TODO 10-16: Pelicula.java

import javax.persistence.*;

@Entity
@Table(name="Pelicula")
public class Pelicula {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    Long idPelicula;
    
    @Column
    String titulo;
    
    @Column
    int anyo;
    
    @ManyToOne(fetch = FetchType.EAGER)
    @JoinColumn(name = "idDirector")
    Director director = null;
}

@Override
public Pelicula savePelicula(Pelicula nuevaPelicula) {
    if (nuevaPelicula != null) {
        return peliculaRepositorio.save(nuevaPelicula);
    }
    return new Pelicula();
}

@Override
public String deletePelicula(Long id) {
    Optional<Pelicula> p = peliculaRepositorio.findById(id);
    if (p.isPresent()) {
        peliculaRepositorio.deleteById(id);
        return "Película eliminada";
    }
    return "La película no existe";
}

@Override
public List<Pelicula> findByDirectorId(Long idDirector) {
    return peliculaRepositorio.findByDirectorId(idDirector);
}

@Override
public List<Pelicula> findPeliculasSinDirector() {
    return peliculaRepositorio.findPeliculasSinDirector();
}

        model.addAttribute("pelicula", pelicula.get());

@RequestMapping(value = "/pelicula/add")
public String addPelicula(Model model) {
    Pelicula p = new Pelicula();
    model.addAttribute("pelicula", p);
    model.addAttribute("directores", directorService.findAllDirector());
    model.addAttribute("nuevo", true);
    return "peliculaForm";
}

// TODO 29: deletePelicula
@GetMapping(value = "/pelicula/delete/{id}")
public String deletePelicula(@PathVariable Long id) {
    peliculaService.deletePelicula(id);
    return "redirect:/pelicula";
}

// TODO 30: updatePelicula
@RequestMapping(value = "/pelicula/update/{id}")
public String updatePelicula(@PathVariable Long id, Model model) {
    Optional<Pelicula> peliculaOpt = peliculaService.findPeliculaById(id);
    if (peliculaOpt.isPresent()) {
        model.addAttribute("pelicula", peliculaOpt.get());
        model.addAttribute("directores", directorService.findAllDirector());
        model.addAttribute("nuevo", false);
        return "peliculaForm";
    }
    return "redirect:/pelicula";
}
